# 🏥 Siham Valetudeo Hospital Website

> **Excellence in Health. Compassion for Life.**

A modern, fully responsive hospital website for **Valetudeo Global Hospital**, built with Tailwind CSS and Alpine.js. The site showcases the hospital's services, medical specialists, pharmacy, nursing, and provides an easy-to-use appointment booking experience.

---

## 🌐 Live Preview

**[View the live site on GitHub Pages →](https://bainemmanuel86-hub.github.io/Siham_Valetudeo_hospital_website/)**

---

## ✨ Features

- **Fully Responsive Design** – Optimised for mobile, tablet, and desktop screens
- **Interactive Doctor Directory** – Search and filter doctors by name or specialty
- **Appointment Booking Form** – Simple, accessible contact & booking form
- **Mobile Navigation Menu** – Hamburger menu powered by Alpine.js
- **Smooth Scrolling** – Section-based navigation with anchor links
- **Speciality Showcase** – Cards highlighting key medical departments
- **Pharmacy & Nursing Sections** – Dedicated sections for non-physician services
- **International Patients Section** – Information for patients travelling from abroad
- **Testimonials Section** – Patient reviews and success stories
- **Leadership & Vision** – Mission, Vision, and Values section with founding team profiles

## 🚀 Getting Started

No build step is required. The site uses CDN-hosted libraries.

### Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bainemmanuel86-hub/Siham_Valetudeo_hospital_website.git
   cd Siham_Valetudeo_hospital_website
   ```

2. **Open in your browser:**
   ```bash
   # Simply open index.html in any modern browser
   open index.html
   ```
   Or use a local development server for the best experience:
   ```bash
   # Using Python
   python3 -m http.server 8080
   # Then open http://localhost:8080
   ```

> ⚠️ An active internet connection is required to load Tailwind CSS, Alpine.js, and Google Fonts from their respective CDNs.
